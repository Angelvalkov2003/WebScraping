# Уеб скрейпинг – новини (GUI)

Приложение с графичен интерфейс за извличане на заглавие, линк към снимка и текст от новинарски статии.

## Инсталация

```bash
cd d:\programirane\WebScraping
pip install -r requirements.txt
```

## Стартиране

```bash
python app_gui.py
```

## Използване

1. **Тип сайт** – изберете от падащото меню (напр. „12 Punto”). В бъдеще могат да се добавят други сайтове.
2. **Линк** – поставете пълния URL на статията (напр. от 12punto.com.tr).
3. Натиснете **Извлечи**.

В полетата ще се покажат:
- **Заглавие** – заглавието на новината
- **Линк към снимка** – URL на главната снимка
- **Текст на новината** – извлеченият текст на статията

## Добавяне на нов тип сайт

1. Създайте нов файл в `scrapers/` (напр. `other_site.py`).
2. Имплементирайте клас, наследяващ `scrapers.base.BaseScraper`, с методи `can_handle(url)` и `scrape(url)`.
3. Регистрирайте го в `scrapers/registry.py` в списъка `SCRAPERS`.

## Зависимости

- Python 3.8+
- `requests` – HTTP заявки
- `beautifulsoup4` – парсване на HTML

GUI използва вградения `tkinter` (има в стандартната инсталация на Python).
